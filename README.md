# Phishing-Website-Detection
A cybersecurity research project leveraging Machine Learning to identify and mitigate emerging phishing threats through automated URL analysis.

##  Project Overview
This project focuses on developing an automated system to identify fraudulent websites by analyzing URL structures and web content features. By leveraging Machine Learning algorithms, the system can distinguish between legitimate and phishing URLs with high precision, mitigating emerging cyber threats.

* **Achievement**: Attained an accuracy of approximately **97%** during the testing phase.
* **Institution**: Final Project at University of Economics and Finance (UEF).

##  Technologies & Tools
* **Programming Language**: Python.
* **Libraries**: Pandas, Scikit-learn (for model implementation and evaluation).
* **Algorithms**: Random Forest, Decision Tree, Multilayer Perceptrons (MLPs), XGBoost Classifier, Autoencoder Neural Network, Support Vector Machines (SVMs).
* **Environment**: Google Colab.

##  Dataset
The model was trained and evaluated using the **PhiUSIIL Phishing Dataset**.
* **Features Analyzed**: URL Length, IsHTTPS, JavaScript usage, and other web content characteristics.
* **Processing**: Performed comprehensive data pre-processing and feature extraction to ensure high model performance.
